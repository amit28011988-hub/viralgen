Automatically posts a motivational meme to your Facebook page **every day at 9 AM IST**.

## üìã Setup Instructions

### Step 1: Add Facebook Token to GitHub Secrets

1. Go to your GitHub repository: https://github.com/amit28011988-hub/viralgen
2. Click **"Settings"** (top menu)
3. Click **"Secrets and variables" ‚Üí "Actions"** (left sidebar)
4. Click **"New repository secret"**
5. Add the following secret:
   - **Name:** `FB_ACCESS_TOKEN`
   - **Value:** (Your Facebook token)
   - Click **"Add secret"**

### Step 2: Get Your Facebook Page ID

Run this command to find your Page ID:
```bash
curl "https://graph.facebook.com/v18.0/me/accounts?access_token=YOUR_TOKEN_HERE"
```

Then add it as another secret:
- **Name:** `FB_PAGE_ID`
- **Value:** (The ID from the response)

### Step 3: Enable GitHub Actions

1. In your repository, go to **"Actions"** tab
2. If prompted, click **"I understand my workflows, go ahead and enable them"**

### Step 4: Test It!

1. Go to **"Actions"** tab
2. Click **"Auto Post to Facebook"** workflow
3. Click **"Run workflow"** ‚Üí **"Run workflow"** (manual trigger)
4. Wait 30 seconds and check your Facebook page!

## üîß Troubleshooting

**If posting fails:**
1. Check that your Facebook token hasn't expired
2. Verify the Page ID is correct
3. Make sure the token has `pages_manage_posts` permission

**Token expires?**
Get a long-lived token:
```bash
curl "https://graph.facebook.com/v18.0/oauth/access_token?grant_type=fb_exchange_token&client_id=YOUR_APP_ID&client_secret=YOUR_APP_SECRET&fb_exchange_token=YOUR_SHORT_TOKEN"
```
